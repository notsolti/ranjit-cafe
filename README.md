# ☕ Ranjit Cafe Website

Official website for **Ranjit Cafe**.

## 📁 Project Structure

```
ranjit-cafe/
├── index.html          # Homepage
├── menu.html           # Menu page
├── about.html          # About us page
├── contact.html        # Contact page
├── gallery.html        # Gallery page
├── css/
│   ├── style.css       # Main styles
│   ├── responsive.css  # Mobile/responsive styles
│   └── animations.css  # Animations
├── js/
│   ├── main.js         # Main JavaScript
│   ├── menu.js         # Menu interactions
│   └── gallery.js      # Gallery logic
└── assets/
    ├── images/         # Photos and images
    ├── icons/          # Icons and favicons
    └── fonts/          # Custom fonts
```

## 🚀 Pages
- **Home** — Hero section, welcome message, highlights
- **Menu** — Food and drinks menu with categories
- **About** — Story, team, values
- **Gallery** — Photos of the cafe
- **Contact** — Location, hours, contact form

## 🛠️ Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
